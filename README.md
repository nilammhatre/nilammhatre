# 📂 Practice Cloud Project

### AWS Serverless Student App

A **learning project** built while exploring **AWS serverless architecture**.

The application uses **Amazon CloudFront for content delivery**, **Amazon S3 to host the static frontend**, and **API Gateway with AWS Lambda** to process API requests, while **DynamoDB stores student data**.

Through this project I understood the **basic request flow of serverless applications and how CloudFront, S3, API Gateway, Lambda, and DynamoDB integrate to build scalable cloud solutions.**
